# ReAmp

ReAmp is a lightweight Winamp-inspired audio player built for Android. It combines nostalgia with modern streaming features, giving you a rich and customizable listening experience.

## Features

- Stream over 20,000 radio stations from around the world  
- MilkDrop-inspired visuals  
- Classic Winamp GUI for a nostalgic experience  
- Play local audio files  
- Add and play custom URLs  
- Sleep timer for automatic shutdown  
- Lightweight and fast
